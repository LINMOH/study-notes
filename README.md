# study-notes

我的初中学习笔记

# 为什么我要整理这个笔记？

无论是在学校，抑或是在家，我都更习惯于纸质学习，因为电子设备容易让我分心。我一直认为自己对学习比较认真，因此笔记也做得很仔细。然而，我不希望将来在查找某条笔记时手忙脚乱。于是，我决定把笔记整理到这里，并在转录的过程中顺便复习。

本项目不会按照年级分类，而是以知识点为单位分类，这样可以保证知识的连贯性。如果这个项目能帮助到别人，那将是我的荣幸。
