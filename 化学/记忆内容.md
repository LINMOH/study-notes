# 八大沉淀

| 化合物       | 化学式          | 沉淀颜色 |
|--------------|----------------|----------|
| 氢氧化铁     | Fe(OH)<sub>3</sub> | 红褐色沉淀 |
| 氢氧化铜     | Cu(OH)<sub>2</sub> | 蓝色沉淀 |
| 氢氧化镁     | Mg(OH)<sub>2</sub> | 白色沉淀 |
| 氢氧化铝     | Al(OH)<sub>3</sub> | 白色沉淀 |
| 碳酸钙       | CaCO<sub>3</sub>   | 白色沉淀 |
| 碳酸钡       | BaCO<sub>3</sub>   | 白色沉淀 |
| 氯化银       | AgCl             | 白色沉淀 |
| 硫酸钡       | BaSO<sub>4</sub>  | 白色沉淀 |
